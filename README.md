## 주요 기능

**🚀 M4 Pro 퀀트 분석**
몬테카를로 미래주가 시뮬레이션(25,000경로 × 252일, 벡터화 NumPy)과 16코어 GPU를 활용한 3D 확률 지형도·변동성 표면(WebGL)을 구현했습니다. Sharpe·Sortino·VaR·CVaR·MDD·CAPM 베타/알파 회귀 같은 리스크 지표를 자동으로 산출하며, ETF는 HHI 집중도·월별 히트맵·DCA 백테스트·3D 상관행렬까지 다룹니다.

**🤖 AI 투자 분석 에이전트**
로컬 LLM(qwen3·gemma)과 Gemini를 연동한 채팅 기능입니다. 뉴스·재무·지배구조·애널리스트 등 7종 도구를 키워드 기반으로 **미리 수집한 뒤** 답하는 결정적(deterministic) 구조로 만들어, AI 환각을 원천에서 차단했습니다. 증권사 리포트 PDF 뷰어와 AI 요약도 포함됩니다.

**🎨 macOS 26 'Liquid Glass' 테마**
반투명 유리 질감, 라이트·다크 테마, 숫자 롤링 애니메이션 등 Apple HIG를 반영했고, PyInstaller로 설치 가능한 DMG로 빌드됩니다.

## 📖 상세 기능 설명서

> 위 대표 화면에 다 담기지 않는 부분까지, 화면별로 무엇을 어떻게 보여주는지 정리했습니다.
> 백엔드는 83개 기능 경로(라우트)로 이루어져 있으며, 아래는 사용자가 실제로 마주하는 화면 단위로 묶은 것입니다.

### 1. 홈 · 통합 검색

앱을 켜면 종목·ETF를 한 칸에서 찾는 검색창과 주요 메뉴 카드(업종 지수, 시장 현황, 세계 시장, 백테스터, 경제 지표, 증권사 리포트)가 보입니다. 검색창에 종목명이나 6자리 코드를 입력하면 자동완성이 종목과 ETF를 구분해 제안하고, 선택하면 해당 리포트가 탭으로 열립니다. 여러 종목을 탭으로 동시에 띄워 비교할 수 있습니다. 상단 우측에는 라이트·다크 테마 토글과 실시간 KOSPI 지수 티커를 두었습니다.

### 2. 개별주식 리포트 (5개 탭)

종목을 검색하면 다음 5개 탭으로 이루어진 리포트가 열립니다.

- **종목 개요** — 실시간 현재가(KRX), 시가총액·등락률·거래량, 시장·업종코드·대표자·설립일·52주 최고/최저·연율 변동성 같은 기본 정보와 기업 개요를 한 화면에 담았습니다.
- **재무제표** — 실적(매출·영업이익), 안정성(부채비율·유동비율·당좌비율), 재무(매출·영업이익·순이익) 세 개 차트를 연간·분기 토글로 전환할 수 있고, 그 아래 DART 사업보고서 기반 손익계산서(최신 분기 포함)를 표로 제공합니다.
- **기업 정보** — 회사 개요, 지배구조·최대주주 등.
- **🚀 M4 퀀트 분석** — 아래 [4번](#4--m4-퀀트-분석-개별주식)에서 따로 설명합니다.
- **✨ AI 코멘터리** — 로컬 LLM이 해당 종목의 퀀트 통계(리스크·수익·밸류에이션)를 읽어 투자 코멘트로 정리합니다. 이때 오늘 날짜와 실데이터를 함께 넣어, 모델이 오래된 기억이 아니라 최신 사실에 근거해 답하도록 했습니다.

### 3. ETF 리포트

ETF를 검색하면 ETF 개요 탭과 M4 퀀트 분석 탭이 열립니다. 개요에는 구성종목·섹터·자산 배분이, 퀀트 탭에는 ETF 전용 분석(아래 [5번](#5--m4-퀀트-분석-etf))이 들어갑니다.

### 4. 🚀 M4 퀀트 분석 (개별주식)

투자론·재무관리에서 배운 분석을 화면으로 옮긴 부분입니다. 탭을 누르면 단계별 진행률(SSE 스트리밍)과 함께 아래 항목을 계산해 보여주며, 화면은 어두운 '콕핏' 테마로 렌더링됩니다.

| 분석 | 내용 |
|---|---|
| 리스크·수익 지표 타일 | Sharpe · Sortino · VaR 95% · CVaR 95% · MDD · 연율 수익률/변동성 · 몬테카를로 1년 VaR |
| 몬테카를로 미래주가 | GBM 모형으로 **25,000경로 × 252일**을 벡터화 계산, 5/25/50/75/95 퍼센타일 부채꼴 분포 |
| 3D 확률 지형도 | SciPy KDE 기반 확률 표면 (WebGL `go.Surface`, 자동회전) |
| 3D 변동성 표면 | 관측창 [5·10·21·42·63·126]일 × 시간축 변동성 표면 |
| 수익률 분포·팻테일 | 히스토그램 vs 정규분포, 왜도·초과첨도 |
| CAPM 베타/알파 | KODEX 200(069500) 대비 일간 로그수익 회귀, R² |
| 프랙탈 패턴 매칭 | 과거 유사 구간 상위 5개 탐색 |

### 5. 🚀 M4 퀀트 분석 (ETF)

ETF는 개별주식과 다른 집중도·배분 중심 지표를 제공합니다.

| 분석 | 내용 |
|---|---|
| 리스크·집중도 타일 | Sharpe · MDD · VaR · **HHI 집중도** · 유효 종목수 · 평균 상관계수 |
| 월별 수익률 히트맵 | 계절성 시각화 |
| 누적 수익 + 언더워터(MDD) | 낙폭 구간 시각화 |
| 롤링 변동성 | 63일 연율 변동성 레짐 |
| 적립식 백테스트 | 매월 정액 투자(DCA) vs 일시불 비교 |
| 3D 편입종목 상관행렬 | 구성종목 간 상관관계 표면 |
| 3D 리스크-리턴-비중 지형 | 버블 크기=비중, 실데이터 산점도 |

### 6. 시장 현황 (국내)

코스피·코스닥 현재 지수와 함께 시가총액 상위 종목을 트리맵으로 보여줍니다(블록 크기는 시총, 색은 등락). 상한가·하한가 포착, 종합 시황과 공시 헤드라인도 같은 화면에 두었습니다.

### 7. 업종 지수

코스피·코스닥 업종별 현재 지수를 등락률 순으로 정렬하고, 업종을 누르면 구성종목 시세로 들어갑니다.

### 8. 세계 시장

미국·유럽·아시아 주요 지수와 환율(원화 기준)을 약 20초마다 자동으로 갱신합니다. 다우·나스닥·S&P 500 같은 주요 지수를 스파크라인 카드로 보여주고, VIX·달러·금·WTI 같은 글로벌 지표도 함께 담았습니다. 국내·미국·글로벌 세 개 뷰로 전환됩니다.

### 9. 경제 지표 (거시)

한국은행 ECOS의 금리·물가·환율과 미국 FRED의 거시지표(미 국채금리·VIX·달러·금·WTI 등)를 한 화면에 모았습니다. 핵심 지표 타일, 시황 영향 해석, 글로벌 지표 그리드로 이루어집니다.

### 10. 백테스터

SMA 교차·모멘텀·RSI 전략을 일봉 기준으로 검증합니다. 종목·기간·파라미터를 설정해 실행하면 가격 차트 위에 매수·매도 시점이 표시되고, 우측에 누적 수익률·CAGR·MDD·Sharpe·Calmar·Profit Factor·승률 같은 성과 지표와 매수보유(Buy&Hold) 대비 비교가 나옵니다. 모든 계산은 로컬에서 이루어집니다.

### 11. 증권사 리포트 + AI 요약 + PDF 뷰어

네이버 금융 6개 카테고리(시황·종목·산업·투자전략·경제·채권)와 한국은행 RSS·금융통화위원회 자료를 목록으로 모았습니다. 각 리포트는 원문 PDF를 앱 안의 뷰어(줌 지원)로 바로 열 수 있고, AI 요약 버튼을 누르면 로컬 LLM이 리포트의 표와 수치를 읽어 핵심을 정리합니다.

### 12. 실시간 트레이딩 데스크

KIS 증권 API를 연동해 실시간 시세와 10단계 호가창, 체결 흐름, 거래량 순위 스크리너를 증권사 HTS 수준으로 구현했습니다. 페이퍼 트레이딩(모의 투자)으로 매수·매도를 시뮬레이션할 수 있으며, 현금 잔고가 정확히 맞아떨어지도록 검증했습니다. 실제 주문은 절대 나가지 않습니다.

### 13. 스크리너

조건을 지정해 종목을 걸러내는 화면입니다(상단 '스크리너' 버튼).

### 14. 해외주식

상단 '해외' 탭으로 전환해 미국·글로벌 종목을 검색하면, 히어로·KPI·차트와 함께 개별주식과 같은 M4 퀀트 분석을 제공합니다.

### 15. ✨ AI 질문하기 (플로팅 채팅)

화면 어디서든 우하단 버튼으로 띄우는 채팅 위젯입니다. 핵심은 지금 보고 있는 화면의 데이터를 읽어 답한다는 점입니다. 종목·ETF·해외·매크로·지수 화면별로 현재 맥락을 인식해, 그 화면의 실데이터 안에서만 답하도록 했습니다. 로컬 LLM(qwen3·gemma)과 Gemini를 고를 수 있고, 추론형과 지시형 모델을 자동으로 구분해 처리하며, 답변은 마크다운으로 렌더링됩니다.

### 16. 🎨 디자인 · 앱 완성도

Apple의 macOS 26 UI Kit(Figma)를 기준으로 토큰을 맞춘 'Liquid Glass' 테마입니다. 반투명 유리 질감과 50px 블러를 적용했고, 숫자는 바뀐 자릿수만 세로로 굴러가도록(오를 때 위로, 내릴 때 아래로) 만들었습니다.

**모든 화면이 라이트·다크 두 테마를 지원합니다.** 상단 토글로 즉시 전환되며, 어느 화면에서 바꿔도 열려 있는 모든 탭에 같은 테마가 동기화됩니다(`kmkt` postMessage). 위 갤러리는 라이트 테마이고, 아래는 같은 종목 개요 화면을 다크 테마로 본 모습입니다.

### 17. 🛠 개발자 모드 — 화면을 클릭하면 소스 위치를 찾아 주는 도구

![개발자 모드](docs/screenshots/18_dev_mode.png)

AI 에이전트와 함께 앱을 빠르게 고치려고 직접 만든 개발 보조 도구입니다. 일반 사용 기능은 아니지만, AI와 어떻게 협업했는지 보여주는 부분이라 함께 적었습니다. 고칠 부분을 말로 설명하면 AI가 다시 코드에서 그 위치를 찾느라 시간을 쓰는 일이 잦았는데, 그 과정을 줄이려고 만들었습니다. 환경변수 `KMKT_DEV=1`로 켤 때만 동작하고(평소에는 완전히 꺼져 화면·성능에 영향이 없습니다), 앱 안에서 `⌘⇧D`로 오버레이를 켜고 끕니다.

동작은 다음과 같습니다.

- **🖱 검사 모드** — 켜면 화면의 어떤 요소든 가리킬 수 있고, 끄면 앱을 평소처럼 조작합니다.
- **클릭하면 위치 캡처** — 고치고 싶은 요소를 누르면, 그 요소가 어느 소스 파일 몇 번째 줄에서 만들어졌는지 자동으로 역추적하고, 현재 CSS 값과 요소 정보를 팝오버로 보여줍니다.
- **메모 + 세션 추가** — 무엇을 어떻게 바꾸고 싶은지 메모를 달아 모읍니다.
- **⌘+클릭 다중 선택** — 여러 요소를 한 번에 골라(화면 위에 번호가 표시됩니다) 묶음으로 처리할 수 있습니다.
- **세션 저장** — 모은 수정 요청을 AI가 그대로 읽을 수 있는 마크다운 파일(`dev_notes/`)로 저장합니다. 에이전트는 이 파일을 읽고 곧장 해당 소스 위치로 가서 작업합니다.
- **미완료 세션 이어가기 · ⌘⇧C 복사** — 끝내지 못한 세션을 다시 불러와 편집하거나, 요소와 소스 정보를 클립보드로 바로 복사할 수 있습니다.

이렇게 해서, 무엇이 잘못됐는지 보는 사람의 눈과 AI에게 줄 정확한 소스 위치를 곧바로 잇도록 했습니다. 처리된 세션은 `dev_notes/done/`으로 옮겨져 다음 작업과 분리됩니다.

## AI 에이전트 협업 체계 — 이 프로젝트의 핵심

이 프로젝트의 진짜 가치는 기능 목록이 아니라, **AI가 실수하는 패턴을 직접 관찰하고 그 실수를 막는 구조를 하나씩 쌓아 올린 과정**에 있습니다. AI를 동료로 두고 함께 일하다 문제가 생길 때마다, 그 동료가 제 몫을 해내도록 돕는 장치를 하나씩 더해 갔습니다.

- **`CODEMAP.md` (코드 내비게이션 인덱스)** — 83개 라우트·템플릿이 *몇 번째 줄에 있는지* 표로 정리한 'AI를 위한 목차'입니다. 전체를 읽는 대신 필요한 줄만 펼쳐 읽게 해 토큰을 아꼈습니다.
- **`smoke_check.py` (회귀 게이트)** — import 성공·엔드포인트 응답·화면 출력 동일성(골든 해시)을 자동으로 검증합니다. `SMOKE PASS ✓`가 떠야만 작업을 완료로 인정합니다.
- **코드베이스 모듈화** — 13,000줄짜리 단일 파일을 핵심 로직 / 화면 템플릿 / 순수 계산 함수 셋으로 분리해, AI가 한 번에 봐야 할 컨텍스트를 작고 명확하게 줄였습니다.
- **`DEBUG_JOURNAL.md` (버그 방지 저널)** — 증상→원인→해결→예방 구조의 룩업 문서입니다. 세션이 바뀌어도 같은 버그를 다시 디버깅하지 않도록 기록을 외부에 쌓았습니다.
- **Developer Mode (`⌘⇧D`)** — 실행 중인 앱에서 화면 요소를 클릭하면 소스 파일과 줄 번호를 역추적해 AI가 읽을 노트로 저장합니다. 사용자의 눈과 에이전트의 입력을 잇는 직접 채널입니다.
- **Claude Code Hooks (자동 회귀 게이트)** — 백엔드를 편집하면 회귀 검사를 강제로 실행하고, 실패하면 턴 종료 자체를 막습니다. 검증을 깜빡하는 실수를 구조적으로 차단한 장치입니다.
- **Reflect System (교정 자동 포착)** — 사용자의 교정 발화를 신뢰도로 점수화해 축적합니다. 백업 → 무결성 검증 → 실패 시 자동 롤백의 3중 안전장치를 거칩니다.
- **`health_check.py` (능동 구조 점검)** — 파일 비대화나 문서-코드 드리프트를 임계치로 감지해 재설계를 먼저 제안합니다. 에이전트가 스스로를 점검하는 구조입니다.

> 위 장치들은 모두 작업 중 실제로 부딪힌 문제(환각, 컨텍스트 한계, 회귀 사고, 같은 버그 반복)에서 출발해, 그 한계를 메우기 위해 하나씩 도입한 것입니다. AI를 동료로 두고 일할 때 필요한 협업 체계를, 사전 지식 없이 문제를 겪어 가며 처음부터 구성했습니다.

## 실행 방법

```bash
# 네이티브 앱 실행 (권장)
uv run application_build/app.py

# 백엔드만 디버그 실행
MI_NO_OPEN=1 MARKET_PORT=8793 uv run scripts/market_dashboard3_realtime.py

# 앱 빌드 (DMG)
cd application_build && ./build.sh
```

> 실시간 데이터 소스(KIS·DART 등)는 각 기관의 API 키가 필요합니다. 키는 `.env` / `API.env`로 주입하며, **저장소에는 포함되지 않습니다.**

## 저장소 구성

```
scripts/           ← 라이브 백엔드(market_dashboard3_realtime.py), 회귀 게이트, 자동화 훅
market_intel/      ← 데이터 수집·분석·리포트 생성 패키지
application_build/ ← pywebview 런처, 빌드 스크립트
docs/              ← CODEMAP(라인맵 인덱스), DEBUG_JOURNAL(증상→해결 룩업)
tests/             ← 순수함수 단위 테스트 + 골든 렌더 기준선
```
