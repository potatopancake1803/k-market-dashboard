# Project Market Dashboard

한국 증시 종합 인텔리전스 및 실시간 대시보드 애플리케이션입니다.

## 폴더 구조
- `data/`: 원본 데이터 및 결과물 저장.
- `docs/`: 프로젝트 관련 문서.
- `scripts/`: 실행 가능한 메인 스크립트 모음.
- `market_intel/`: 데이터 수집, 분석 및 리포트 생성을 위한 핵심 패키지.

## 실행 방법
```bash
# uv를 이용한 단일 스크립트 실행 (권장)
uv run scripts/market_dashboard3_realtime.py
```
